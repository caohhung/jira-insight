# JIRA INSIGHT

### Require ruby gems:

```
 - jira-ruby
 - gooddata
```

### Important environment variables:

```
 - JIRA_USER
 - JIRA_SECRET
 - GOODDATA_USER
 - GOODDATA_PASSWORD
 - GOODDATA_HOST
 - PROJECT_GROUP
```
